# <p align="center">  bibliophile #

**bibliophile** - An application for searching for words or phrases in the text of various works.

### Technology stack ###
* Python 3.8
* fastapi - web framework for building APIs
* SQLAlchemy - ORM
* PostgreSQL - DataBase<br>
